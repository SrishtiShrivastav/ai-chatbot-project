# 🤖 AI Chatbot (React + FastAPI + Gemini)

## 📌 Project Overview

This is a simple AI chatbot application built using:

* FastAPI (Backend)
* HTML/React UI (Frontend)
* Gemini API for AI responses

## 🚀 Features

* Chat with AI bot
* FastAPI backend integration
* Simple frontend UI
* API-based response system

## 🛠️ Tech Stack

* Python (FastAPI)
* HTML / React (Frontend)
* Gemini API

## ▶️ How to Run

1. Install requirements:
   pip install -r requirements.txt

2. Run backend:
   uvicorn main:app --reload

3. Open frontend:
   Open index.html in browser

## 📷 Output

Chat interface where users can send messages and receive AI responses.
